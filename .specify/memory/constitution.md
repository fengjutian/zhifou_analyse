# 《知否》电视剧分析文章 Constitution

## Core Principles

### 1. 结构清晰
- 必须包含：引言、剧情脉络、人物分析、冲突机制、主题提炼、结论。
- 段落层次清楚，避免过长段落。
- 先总后分，再回到总。

### 2. 论证可检验
- 每个结论必须对应至少一个剧情事实或人物行为。
- 避免空泛判断，优先用“场景 + 行动 + 结果”。
- 关键观点要指出发生阶段或情节位置。

### 3. 人物分析一致性
- 主角与关键配角分别给出：初始状态、转折事件、能力变化、最终定位。
- 不允许前后自相矛盾的性格描述。
- 角色关系必须与剧情事实对齐。

### 4. 主题提炼可落地
- 主题必须回到具体冲突与解决机制。
- 主题描述采用“价值张力 + 解决路径”的句式。
- 禁止只给抽象口号而无剧情支撑。

### 5. 语言与风格
- 用词准确，避免网络化空洞表达。
- 允许少量文学化修辞，但必须服务论证。
- 统一称谓与术语，全文一致。

## 产出标准
- 输出包含至少一份表格，建议使用：剧情时间线表、人物成长表、冲突-解决矩阵。
- 全文应能直接用于答辩或PPT，结尾提供要点式总结。
- 术语表可选，但如出现自定义概念必须解释。

## 质量门槛
- 逻辑跳跃、无证据判断、人物事实错误视为不可接受。
- 发现错误须修订并保留改动理由。
- 任何新增结论必须补充支撑证据。

## Governance
- 本文件优先于其他写作习惯。
- 例外情况需在文末注明理由。
- 任何改动需记录修订时间与版本。
