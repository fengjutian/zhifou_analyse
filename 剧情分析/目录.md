# 剧情分析目录

## 阅读顺序建议
按 `故事概述 -> 人物分析 -> 情节分析 -> 主题分析` 阅读，可形成从“发生了什么”到“为什么这样发生”的完整理解链路。

## 章节导航

1. [01_故事概述.md](./01_故事概述/01_故事概述.md)  
聚焦全剧主线与阶段推进，概括从盛家内宅到顾家侯府的叙事升级。

2. [02_人物分析.md](./02_人物分析/02_人物分析.md)  
拆解核心人物与群像关系，说明人物如何通过选择与关系治理推动剧情。

3. [03_情节分析.md](./03_情节分析/03_情节分析.md)  
按冲突类型梳理关键矛盾、推进机制与解决路径，展示剧情递进逻辑。

4. [04_主题分析.md](./04_主题分析/04_主题分析.md)  
提炼女性成长、礼法张力与家族结构等核心主题，给出全剧价值落点。
